在本书前面的章节中，我们系统地探索了一系列旨在提高 C++ 代码质量的自动化解决方案。其中包括采用清晰的命名约定、利用现代 C++ 功能、严格实施单元和端到端测试、维护干净的提交和消息以及有效使用 git bisect 等调试技术等。这些实践中的每一种都是我们编写健壮、可维护软件工具包中的重要组成部分。
